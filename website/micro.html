微观技能溢价分析

数据来源：BOSS直聘 (Zhipin.com)

1. NLP 技能解码与特征工程

我们利用自然语言处理技术对职位描述（JD）进行深度解析。通过接入阿里通义千问 API (Qwen-Plus)，我们成功剔除了行政废话，提取了真正决定薪资的“硬技能”。

图 1-0: 交互式技能词云/钻石图 (BOSS直聘数据)

2. 薪资决定因素分析

通过相关性热力图和特征重要性排序，我们探究了学历、经验、技能相似度等因素对薪资的影响权重。

图 2-0: 交互式薪资流向桑基图 (BOSS直聘数据)

3. 技能溢价测度

本研究的核心发现：职位描述（JD）与高薪原型的相似度越高，其提供的薪资溢价越显著。这验证了劳动力市场对特定技能组合的高回报。

图 3-0: 交互式技能溢价回归散点图 (BOSS直聘数据)

图 3-0.5: 交互式高低薪人才画像雷达图 (BOSS直聘数据)

4. 异质性分析

我们进一步探讨了技能溢价在不同学历、城市等级等维度上的差异。

图 4-0: 交互式城市薪资地图 (BOSS直聘数据)

5. 稳健性检验

为了确保结论的科学性，我们进行了倾向得分匹配 (PSM) 和安慰剂检验。
